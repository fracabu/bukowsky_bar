<a name="english"></a>
## :gb: English

### Overview

Sophisticated dark-themed website for Bukowski's Bar, a gallery/cafe/bistrot in Borgo Pio, Rome. 1920s speakeasy aesthetic with modern functionality.

### Features

- **Dark Speakeasy Theme** - 1920s vintage aesthetic with gold accents
- **Interactive Menu** - Tab-based navigation with 12+ Spritz varieties
- **Gin del Marchese Page** - Dedicated showcase for house gin
- **Contact Form** - With Google Maps integration
- **Quick Actions** - One-tap call, WhatsApp, directions

### Pages

| Page | Status | Description |
|------|--------|-------------|
| Homepage | Complete | Hero, About, Experience cards |
| Il Nostro Gin | Complete | Botanicals, Tasting notes, Recipes |
| Menu | Complete | Spritz, Cocktails, Food, Wine |
| Contatti | Complete | Form, Map, Quick actions |

### Design System

- **Colors**: Ink Black, Dark Sepia, Antique Gold, Paper Cream
- **Fonts**: Playfair Display, Cormorant Garamond, Lora, Montserrat
- **Effects**: Fade-in animations, parallax scrolling

### Quick Start

```bash
git clone https://github.com/fracabu/bukowsky_bar.git
cd bukowsky_bar
# Open index.html or use local server
python -m http.server 8000
```
